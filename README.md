### About SamaneMVC
---
SAMANEMVC is a PHP framework based on MVC model (Model View Controller)

### REQUIREMENTS 
The requirements needed to run this frameWork : 
```
    php >= 7.* 
    Mysql 
    Composer (for dependencies) 
 
```
